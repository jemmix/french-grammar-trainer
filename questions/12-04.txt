RULE 12-04: Le pronom relatif où — complément de temps (le jour où je suis arrivé)
Generated by: sonnet
Total: 20 MCQ + 5 input = 25 questions

BEGIN QUESTION
ID: 12-04-001
TYPE: MCQ
PROMPT: Complétez : « Le jour ___ je suis arrivé, il pleuvait. »
RIGHT ANSWER: où
EXPLANATION: « où » est le pronom relatif correct pour exprimer le temps. Il remplace l'antécédent « jour » qui est un complément de temps.
WRONG ANSWER: que
EXPLANATION: « que » est un pronom relatif pour les compléments d'objet direct, pas pour le temps. « Le jour que je suis arrivé » est incorrect.
WRONG ANSWER: qui
EXPLANATION: « qui » est pour les sujets, pas pour les compléments de temps. « Le jour qui je suis arrivé » est grammaticalement incorrect.
WRONG ANSWER: lequel
EXPLANATION: « lequel » est un relatif pour les lieux ou les objets, pas pour les compléments de temps dans ce contexte.
END QUESTION
