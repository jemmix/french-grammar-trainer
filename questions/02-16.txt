ID: 02-16-025
TYPE: INPUT
PROMPT: Remplissez les blancs avec le passé composé pour exprimer une succession d'actions.
PHRASE: « Tout à coup, l'orage ___ , les enfants ___ dedans, je ___ les fenêtres et nous ___ tous ensemble. »
RIGHT ANSWER: a éclaté / sont entrés / ai fermé / nous sommes réunis
EXPLANATION: Le passé composé pour tous les verbes crée une succession d'événements abruptement liés : éclater (avoir), entrer (être), fermer (avoir), se réunir (être).
WRONG ANSWER: éclatait / entraient / fermais / nous réunissions
EXPLANATION: L'imparfait n'exprime pas une succession spécifique d'actions complétées, mais plutôt un contexte habituel ou continu.
WRONG ANSWER: a éclaté / sont entrés / ferme / nous sommes réunis
EXPLANATION: L'incohérence temporelle entre le passé composé (« a éclaté », « sont entrés ») et le présent (« ferme ») puis passé composé est incorrecte.
WRONG ANSWER: éclatera / entreront / fermerai / nous réunirons
EXPLANATION: Le futur n'est pas approprié. Les événements décrits sont passés et complétés, pas futurs.
WRONG ANSWER: avait éclaté / étaient entrés / avais fermé / nous étions réunis
EXPLANATION: Le plus-que-parfait serait inapproprié pour cette succession directe d'actions dans une narration au passé composé.
WRONG ANSWER: éclaterait / entreraient / fermerais / nous réunirions
EXPLANATION: Le conditionnel exprimerait une hypothèse ou une possibilité, pas des actions réellement complétées et successives.
